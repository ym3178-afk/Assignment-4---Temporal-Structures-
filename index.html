Digital Object 2: Temporal Structures

This website preserves the complete original exercise structure. Five D3 canvases move from foundational axes to event timelines, concurrent programs, an externally loaded CSV timeline, and an advanced calendar heatmap. All five canvases now use one unified visual system: consistent margins, typography, line weight, animation, tooltip behavior, and color logic.

01
D3.js Foundation: Basic Setup and Axes

A foundational diagram demonstrating SVG creation, linear scales, axes, labels, margins, and the coordinate system that supports every later visualization.

02
Event Timeline: Extreme Weather Signals

An event-based timeline using temporal position, color, scale, hover interaction, and tooltips to reveal the rhythm and severity of major climate events.

03
Concurrent Events: Community Programs

A Gantt-style visualization showing overlapping community initiatives. Duration, category, progress, and concurrency are visible within a single shared time structure.

04
CSV Timeline: Urban Climate Adaptation

The original 15-record CSV-driven Gantt example now uses a different background dataset: the historical development of urban environmental adaptation. The CSV format remains exactly name,start,end,category.

Run through a local server

Use VS Code Live Server because browsers restrict local CSV loading when an HTML file is opened directly.

CSV structure preserved

name,start,end,category
Only the records and visual interpretation have changed.

05
Advanced D3 Study: Urban Heat Calendar

A second temporal visualization preserves all 754 original daily records across 2021–2023 and replaces only their values with a synthetic urban heat-stress dataset. Every cell represents one weekday; near-baseline days recede into a neutral paper tone while genuine anomalies carry the color — cool violet below the baseline, lime rising to orange under extreme heat.

PRECEDENT STUDY · FORENSIC ARCHITECTURE / FORENSIS
THE BORDER
MOVES IN TIME

In The Evros/Meriç River: A Century of Border Design, time is not a neutral background. Legal decisions, river movement, fence construction, strandings, and wildfire operate at different speeds. The following four visualizations distinguish verified historical records from interpretive analysis.

07
VERIFIED TIMELINE
A Century of Documented Turning Points

This timeline uses only source-supported years or dates, from the 1923 Lausanne Treaty through the 2023 wildfire and the 2024 end of the river-shift study.

08
VERIFIED DURATION CHART
Slow Processes and Short Crises

The chart only includes periods with explicit ranges. Unsupported hydraulic or military start dates are deliberately excluded.

CSV structure: name,start,end,category

INTERPRETIVE EVENT SEQUENCE
From Legal Line to Lethal Condition

This diagram translates the precedent's argument into an analytical sequence. Interpretive nodes are marked separately from source-supported claims.

BINARY CALENDAR HEATMAP
Seventeen Continuous Days of Fire

How Temporal Structure Enters My Project

Through D3.js timelines, Gantt charts, event sequences, and calendar heatmaps, I can connect long environmental transformations with sudden political and humanitarian crises. The project therefore communicates not only what the border is, but how it is repeatedly produced, reinforced, experienced, and contested over time.
